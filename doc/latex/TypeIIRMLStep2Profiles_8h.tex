\doxysubsection{Detailed Description}
\mbox{\hyperlink{structHeader}{Header}} file for the Step 2 motion profiles. 

\mbox{\hyperlink{structHeader}{Header}} file for the enumeration of all possible Step 2 motion profiles used for within in the library of the Type II On-\/\+Line Trajectory Generation algorithm. Further more, this file contains all calculation functions of the Step 2 motion profiles. These functions calculate the all trajectory parameters for a specific motion profile.

The enumeration as well as all profile functions are part of the namespace Type\+I\+I\+R\+M\+L\+Math.
