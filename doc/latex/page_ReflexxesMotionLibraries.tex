\hypertarget{page_ReflexxesMotionLibraries_sec_IntroductionOverview}{}\doxysection{Introduction}\label{page_ReflexxesMotionLibraries_sec_IntroductionOverview}
The {\itshape Reflexxes Motion Libraries} are designed to achieve new opportunities in sensor-\/based motion control opening the door to new applications, safe human-\/robot interaction, and advanced robot motion capabilities. The three key features of these libraries are\+:


\begin{DoxyItemize}
\item {\bfseries{Jerk-\/limited robot motions can be calculated from arbitrary initial states of motion (i.\+e., during any motion).}}
\item {\bfseries{New motions are calculated within one low-\/level control cycle (typically within one millisecond or less).}}
\item {\bfseries{The interface is very simple and clear, such that it can easily be integrated in existing systems.}}
\end{DoxyItemize}

Based on the Reflexxes framework, the following new motion control features can be realized with the {\itshape Reflexxes Motion Libraries}\+:


\begin{DoxyItemize}
\item {\bfseries{Instantaneous changes of coordinate frames}}
\item {\bfseries{Instantaneous changes of control state spaces}}
\item {\bfseries{Deterministic, instantaneous reactions to sensor signals}}
\item {\bfseries{Safe and stable reactions to sensor failures}}
\item {\bfseries{Simple and robust visual servo control}}
\item {\bfseries{Stable switched-\/system control}}
\end{DoxyItemize}

All these features let robots and mechanical systems {\bfseries{instantaneously}} react to {\bfseries{unforeseen}} events and enable new opportunities for sensor-\/based robot motion control --- and and thus helping to realize new and advanced robot and motion control applications.~\newline
