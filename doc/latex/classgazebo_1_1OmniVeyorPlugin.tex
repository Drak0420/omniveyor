\doxysubsection{Detailed Description}
An abstract gazebo model plugin for tracked vehicles. 

\begin{DoxySince}{Since}
8.\+1
\end{DoxySince}
The plugin processes the following parameters (all have defaults)\+: $<$max\+\_\+linear\+\_\+speed$>$ Max linear velocity in m/s. Also max track velocity. Default is 1.\+0. $<$max\+\_\+angular\+\_\+speed$>$ Max angular speed in rad/s. Default is 1.\+0. $<$robot\+\_\+namespace$>$ Namespace used as a prefix for gazebo topic names. Default is the name of the model.
