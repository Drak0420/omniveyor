The {\itshape position-\/based Type II On-\/\+Line Trajectory Generation algorithm} called at a time instant $ T_i $ computes the following output values (cf. \mbox{\hyperlink{classRMLPositionOutputParameters}{R\+M\+L\+Position\+Output\+Parameters}})\+:~\newline
~\newline
 
\begin{DoxyItemize}
\item The state of motion $ {\bf M}_{i+1} $ at time instant $ T_{i+1} $ consisting of~\newline
~\newline
 
\begin{DoxyItemize}
\item the new position/pose vector $ \vec{P}_{i+1} \ \Longrightarrow \ $ \mbox{\hyperlink{classRMLOutputParameters_aad28c60ca31e3c86440d4f0caaca546e}{R\+M\+L\+Position\+Output\+Parameters\+::\+New\+Position\+Vector}},~\newline
~\newline
 
\item the new velocity vector $ \vec{V}_{i+1} \ \Longrightarrow \ $ \mbox{\hyperlink{classRMLOutputParameters_a1761b13fb525a62e1843b4d58ec571d4}{R\+M\+L\+Position\+Output\+Parameters\+::\+New\+Velocity\+Vector}}, and~\newline
~\newline
 
\item the new acceleration vector $ \vec{A}_{i+1} \ \Longrightarrow \ $ \mbox{\hyperlink{classRMLOutputParameters_a838d1e65d5694cee748edb1919f5e724}{R\+M\+L\+Position\+Output\+Parameters\+::\+New\+Acceleration\+Vector}}.~\newline
~\newline
 
\end{DoxyItemize}
\item The value of the synchronization time $ t_i^{\,sync} $ in seconds (\mbox{\hyperlink{classRMLOutputParameters_a273ecb1a6e080466af95c5851f12f8ca}{R\+M\+L\+Position\+Output\+Parameters\+::\+Synchronization\+Time}}), that is, the time which is required until all selected degrees of freedom reach their desired target state of motion $ {\bf M}_i^{\,trgt} $.~\newline
~\newline
  
\item A boolean flag \mbox{\hyperlink{classRMLOutputParameters_ac5bc428b2a2b6cb8ed4e9a7f9464f4e6}{R\+M\+L\+Position\+Output\+Parameters\+::\+A\+New\+Calculation\+Was\+Performed}}, which indicates, what an entirely new computation was performed for the call of \mbox{\hyperlink{classReflexxesAPI_a30d3cdba072553a5d53aa4ce4b4a77d1}{Reflexxes\+A\+P\+I\+::\+R\+M\+L\+Position()}} (cf. \mbox{\hyperlink{page_RealTimeBehavior}{Real-\/\+Time Behavior}}).~\newline
~\newline
 
\item A boolean flag \mbox{\hyperlink{classRMLOutputParameters_a07df860870497955b460cf9e9ee130c5}{R\+M\+L\+Position\+Output\+Parameters\+::\+Trajectory\+Is\+Phase\+Synchronized}}, which indicates, whether the current trajectory is phase-\/synchronized (cf. \mbox{\hyperlink{page_SynchronizationBehavior}{Synchronization Behavior}}).~\newline
~\newline
 
\item All data about the positional extremum values that will occur if the motion trajectory is executed. This includes two vectors, which contain the positional extremum values\+:~\newline
~\newline
 
\begin{DoxyItemize}
\item the minimum positions for each degree of freedom $ \overrightarrow{\mbox{min}}\Big(\vec{p}_i\left(t\right)\Big) \ \Longrightarrow \ $ \mbox{\hyperlink{classRMLOutputParameters_ab09baf1ceba18244ac62718b6770faac}{R\+M\+L\+Position\+Output\+Parameters\+::\+Min\+Pos\+Extrema\+Position\+Vector\+Only}} and~\newline
~\newline
 
\item the minimum positions for each degree of freedom $ \overrightarrow{\mbox{max}}\Big(\vec{p}_i\left(t\right)\Big) \ \Longrightarrow \ $ \mbox{\hyperlink{classRMLOutputParameters_a1f45485fb851432786dba445d3ac9601}{R\+M\+L\+Position\+Output\+Parameters\+::\+Max\+Pos\+Extrema\+Position\+Vector\+Only}}.~\newline
~\newline
 ~\newline
 
\end{DoxyItemize}Furthermore, the times in seconds, at which the positional extremum values are reached, are stored in~\newline
~\newline
 
\begin{DoxyItemize}
\item \mbox{\hyperlink{classRMLOutputParameters_a5cd2864b6303e692422488b625ad06e6}{R\+M\+L\+Position\+Output\+Parameters\+::\+Min\+Extrema\+Times\+Vector}} and~\newline

\item \mbox{\hyperlink{classRMLOutputParameters_a83943264635585ddc1317a27617097bc}{R\+M\+L\+Position\+Output\+Parameters\+::\+Max\+Extrema\+Times\+Vector}}.~\newline
