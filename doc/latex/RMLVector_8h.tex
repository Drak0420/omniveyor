\doxysubsection{Detailed Description}
\mbox{\hyperlink{structHeader}{Header}} file for the dynamic vector class used for the Reflexxes Motion Libraries. 

This file implements a minimalist version of a dynamic vector class to be used for the interface of the Reflexxes On-\/\+Line Trajectory Generation algorithm. Furthermore, three type-\/specific versions of this class are defined\+:


\begin{DoxyItemize}
\item R\+M\+L\+Double\+Vector for {\ttfamily double} values,
\item R\+M\+L\+Int\+Vector for {\ttfamily integer} values, and
\item R\+M\+L\+Bool\+Vector for {\ttfamily bool} values.
\end{DoxyItemize}
