\mbox{\hyperlink{structHeader}{Header}} file for intermediate profile segments of Step 2. 

\mbox{\hyperlink{structHeader}{Header}} file for trajectory execution time calculations of intermediate velocity profiles in the second step of the Type II On-\/\+Line Trajectory Generation algorithm. All profile functions are part of the namespace Type\+I\+I\+R\+M\+L\+Math.
