Three arrays of \mbox{\hyperlink{classTypeIIRMLMath_1_1TypeIIRMLPolynomial}{Type\+I\+I\+R\+M\+L\+Math\+::\+Type\+I\+I\+R\+M\+L\+Polynomial}}. 

This data structure contains three arrays of polynomials required for the Type II On-\/\+Line Trajectory Generation algorithm.
