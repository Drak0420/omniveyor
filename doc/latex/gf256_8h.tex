Initialize a context, filling in the tables.

Thread-\/safety / Usage Notes\+:

It is perfectly safe and encouraged to use a \mbox{\hyperlink{structgf256__ctx}{gf256\+\_\+ctx}} object from multiple threads. The gf256\+\_\+init() is relatively expensive and should only be done once, though it will take less than a millisecond.

The \mbox{\hyperlink{structgf256__ctx}{gf256\+\_\+ctx}} object must be aligned to 16 byte boundary. Simply tag the object with G\+F256\+\_\+\+A\+L\+I\+G\+N\+ED to achieve this.

Example\+: static G\+F256\+\_\+\+A\+L\+I\+G\+N\+ED \mbox{\hyperlink{structgf256__ctx}{gf256\+\_\+ctx}} The\+G\+F256\+Context; gf256\+\_\+init(\&\+The\+G\+F256\+Context, 0);

Returns 0 on success and other values on failure.
