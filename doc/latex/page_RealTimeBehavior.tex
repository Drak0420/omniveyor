The On-\/\+Line Trajectory Generation algorithms (\mbox{\hyperlink{classOTG}{O\+TG}}) of the Reflexxes Motion Libraries are {\itshape real-\/time capable}, that is, {\bfseries{a worst-\/case execution time will not be exceeded}} in any case. To achieve this, the two basic methods of the Reflexxes A\+PI (cf. \mbox{\hyperlink{page_ImportantClasses}{References to All Classes of the Reflexxes A\+PI}})~\newline

execute a limited number of operations. In all cases, the worst-\/case execution time scales linearly with the number of selected degrees of freedom, such that a complexity of $ O(n) $ results, where $ n $ is the number of degrees of freedom.
