\hypertarget{page_SourceCode_sec_InterfaceLayer}{}\doxysection{The Interface Layer}\label{page_SourceCode_sec_InterfaceLayer}
The {\itshape interface layer} is based on the class \mbox{\hyperlink{classReflexxesAPI}{Reflexxes\+A\+PI}} and constitutes the user A\+PI of the Reflexxes Motion Libraries. It wraps around the algorithmic classes \mbox{\hyperlink{classTypeIIRMLPosition}{Type\+I\+I\+R\+M\+L\+Position}} and \mbox{\hyperlink{classTypeIIRMLPosition}{Type\+I\+I\+R\+M\+L\+Position}} (cf. \mbox{\hyperlink{page_SourceCode_sec_AlgorithmLayer}{The Algorithm Layer}}) and hides all functionalities that are not required by the user application in order to obtain a clean and minimal interface for all applications. For a description of input and output values, please refer to the \mbox{\hyperlink{page_InputValues}{Description of Input Values}} and the page\+\_\+\+Output\+Values.~\newline

\hypertarget{page_SourceCode_sec_AlgorithmLayer}{}\doxysection{The Algorithm Layer}\label{page_SourceCode_sec_AlgorithmLayer}
The {\itshape algorithm layer} contains the actual Type II On-\/\+Line Trajectory Generation (\mbox{\hyperlink{classOTG}{O\+TG}}) algorithm. The position-\/based algorithm is realized by the class \mbox{\hyperlink{classTypeIIRMLPosition}{Type\+I\+I\+R\+M\+L\+Position}}, and the velocity-\/based algorithm is realized by the class \mbox{\hyperlink{classTypeIIRMLVelocity}{Type\+I\+I\+R\+M\+L\+Velocity}}. Both classes make use of the namespace Type\+I\+I\+R\+M\+L\+Math, which provides all mathematical foundation for the algorithms (cf. \mbox{\hyperlink{page_SourceCode_sec_MathLayer}{The Math Layer}}). The methods to call the actual algorithms are \mbox{\hyperlink{classTypeIIRMLPosition_af2a8f0d8c55c3f5a4861149a7bc35d66}{Type\+I\+I\+R\+M\+L\+Position\+::\+Get\+Next\+State\+Of\+Motion()}} and \mbox{\hyperlink{classTypeIIRMLPosition_af2a8f0d8c55c3f5a4861149a7bc35d66}{Type\+I\+I\+R\+M\+L\+Position\+::\+Get\+Next\+State\+Of\+Motion()}}, both of which are used by the A\+PI class \mbox{\hyperlink{classReflexxesAPI}{Reflexxes\+A\+PI}} (cf. \mbox{\hyperlink{page_SourceCode_sec_InterfaceLayer}{The Interface Layer}}).
