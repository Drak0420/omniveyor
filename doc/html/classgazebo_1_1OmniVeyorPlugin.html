Detailed Description

An abstract gazebo model plugin for tracked vehicles.

Since
8.1

The plugin processes the following parameters (all have defaults): <max_linear_speed> Max linear velocity in m/s. Also max track velocity. Default is 1.0. <max_angular_speed> Max angular speed in rad/s. Default is 1.0. <robot_namespace> Namespace used as a prefix for gazebo topic names. Default is the name of the model.
