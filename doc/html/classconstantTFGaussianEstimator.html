int  windowLength

Construct a new constant TF Gaussian Estimator object.

The object assumes a near-constant transform to be a white noise process, and estimates its covariance.

Parameters
targetFrameTarget frame of the transformation, such that $_{target}T^{origin} * _{origin}X = _{target}X.$
originOrigin frame of the transform.
tfBufferPointer to an initialized tf2_ros::Buffer, with its tf2_ros::TransformListener already attached.
windowLengthHow many consecutive TF are collected over time to calculate the covariance.

◆ update()

bool constantTFGaussianEstimator::update ( )

Update the internal states: push the latest TF to the record and update the mean over time.

This function has to be called every cycle before other latest* variables are updated.

Returns
true - A newer TF has been updated to the record.
false - No newer TF is available. Last values are maintained.
